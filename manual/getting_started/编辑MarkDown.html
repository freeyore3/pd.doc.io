推荐使用 VSCode

插件安装

推荐在 VSCode 中安装如下两个插件：

  • Markdown All in One
    • 快捷键 Ctrl+Shift+V 打开Preview面板，实时查看
  • Paste Image
    • 设置
      • 打开 VS Code 的设置（快捷键为 Ctrl + , 或者 Cmd + ,）。
      • 在设置中搜索 "paste image"。
      • 找到 "Paste Image: Path" 选项，将其设置为 ${projectRoot}/images。这会告诉插件将图片保存在项目根目录下的 images 文件夹内。
    • 快捷键
      • 在 Markdown 文件中，使用快捷键（默认为 Ctrl + Alt + V 或者 Cmd + Alt + V）粘贴图片

打开工作区目录

VSCode 打开目录 PDClient/Doc 作为根目录

Note

这样使用Paste Image粘贴的截图可以正确保存在 PDClient/Doc/images 目录中
