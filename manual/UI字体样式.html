① 字体样式配置

a、字体模板配置

1、在这个目录下配置好字体模板的预制体，该步骤由UE操作， 相应的操作规范在这里 2、配置好后UI需要过一遍效果是否正确, 不正确返回第一步 3、QA再看下配置是否正确导入。不正确返回第一步， 如果找不到问题错在哪，找相关程序查询

b、字体模板使用

当模板配置好之后，就可以在LanguageI18nText组件上看到了， 如下图所示:

总共有以下几种情形:

1、静态文本，静态字体模板

UE直接在预制内配置好即可

2、静态文本，动态字体模板

UE在预制内配置好多语言key，动态字体模板需要程序根据业务设置，代码如下所示:

label.SetFontStyle(FontStyleDef.***);

// 在LanguageI18nText有相应的接口
public void SetFontStyle(FontStyleDef newStyle) => SetFontStyle(newStyle, false);

3、动态文本，静态字体模板

UE配置好字体模板，如果需要格式化的话，UE需要在预制内配置好格式化选项 程序根据业务实际情况设置文本，如果发现与UE设置不一致，根据情况同步UE

4、动态文本，动态字体模板

UE配置好基础项即可，其他交由程序处理

c、富文本

富文本有几种情况大致如下:

1、静态

直接在多语言内配置好即可

2、动态

需要程序根据业务设置不同模板
