PD

说明

  • 文档统一放在manual目录下，markdown格式，支持用文件夹分类，目录会自动生成

MarkDown语法参考

建议在 vs code 安装MarkDown插件进行编辑

安装环境

命令行运行检查是否已安装 dotnet

dotnet --version

dontnet官方网站

安装好 dotnet 之后，运行 install.bat，即可

Build

生成manual

运行 build_site.bat，自动更新目录

生成api

运行 build_api.bat，自动生成Unity工程中Framework、HotUpdate两个asmdef的API参考

代码注释格式如下，将自动生成到api帮助文档

namespace MyNamespace
{
    /// <summary>
    /// Enter description here for class X.
    /// ID string generated is "T:MyNamespace.MyClass".
    /// </summary>
    public unsafe class MyClass
    {
        /// <summary>
        /// Enter description for method someMethod.
        /// ID string generated is "M:MyNamespace.MyClass.someMethod(System.String,System.Int32@,System.Void*)".
        /// </summary>
        /// <param name="str">Describe parameter.</param>
        /// <param name="num">Describe parameter.</param>
        /// <param name="ptr">Describe parameter.</param>
        /// <returns>Describe return value.</returns>
        public int someMethod(string str, ref int nm, void* ptr) { return 1; }
    }
}

全部生成

运行 build_all.bat，生成manual和api

查看文档

运行 run.bat，将打开网页定位到http://127.0.0.1:8081/
